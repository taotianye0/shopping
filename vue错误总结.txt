
1、引入方式不对
exprot defult {  } 暴露的变量 =====》 引入方式 import 自定义变量名 from "...路径";
exprot const。。。 暴露的多个变量 =====》 引入方式 import {自定义变量名1，自定义变量名2} from "...路径";

2、请求方式不对
要认真看后端API的要求，参数要求是什么？ 有哪些参数？没有的参数自己补齐，参数值不对的进行修改===》提交之前一定要提前看看值

3、传参方式不对

后端要求：
| 参数名 | 说明     |
| :----- | -------- |
| id     | 菜单编号 |

传参数的时候应该写为json形式  ===》 {id ：变量名}

4、使用element-ui修饰页面的时候属性使用方式不对
官网地址：https://element.faas.ele.me/#/zh-CN
先查看是什么标签再根据标签去官网查看相应的属性使用方式
